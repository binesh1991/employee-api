# How to run

## React app

1.  Open the React\employee-list in a command prompt.
2.  Use "npm start" to run the app.

## DotNet app

1.  Open the DotNet\EmployeeList\EmployeeList.sln file in Visual Studio.
2.  Run/debug using the "http" mode.

# Test plan
